# C Programming Repository

This repository contains a collection of C programming codes developed as part of academic practice and learning.

# Contents
- Searching Algorithms (Linear Search, Binary Search)
- Array Operations
- File Handling Programs
- Dynamic Memory Allocation
- Recursion Programs

# Objective
The purpose of this repository is to understand and implement fundamental concepts of C programming and improve problem-solving skills.

# How to Run
1. Compile the program using a C compiler (e.g., gcc filename.c)
2. Execute the compiled file (e.g., ./a.out)
